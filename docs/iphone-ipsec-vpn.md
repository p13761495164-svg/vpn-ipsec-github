# 在 iPhone 上配置 IPSec VPN

本教程适用于 iPhone 自带的 **IPSec（Cisco IPSec）** 连接。开始前，请向 VPN 服务商或网络管理员取得以下信息：

- 服务器地址
- 用户名和密码
- 群组名称（如有）
- 共享密钥（Secret）或证书

> 不要把密码或共享密钥发给他人，也不要保存到公开的 GitHub 仓库。

## 配置步骤

1. 打开 **设置**。
2. 进入 **通用** → **VPN 与设备管理** → **VPN**。
3. 点按 **添加 VPN 配置…**。
4. 点按 **类型**，选择 **IPSec**。
5. 填写配置：

   | 项目 | 填写内容 |
   | --- | --- |
   | 描述 | 自定义名称，例如 `公司 VPN` |
   | 服务器 | VPN 服务器的域名或 IP 地址 |
   | 账户 | VPN 用户名 |
   | 密码 | VPN 密码 |
   | 使用证书 | 使用共享密钥时保持关闭；使用证书时按管理员要求设置 |
   | 群组名称 | 填写管理员提供的群组名称；没有则留空 |
   | 密钥 | 填写管理员提供的共享密钥 |

6. 点按右上角 **完成**。
7. 返回 VPN 页面，选中刚添加的配置，然后打开 **VPN 状态** 开关。
8. 屏幕顶部出现 VPN 标志，或状态显示“已连接”，即表示连接成功。

## 连接不上怎么办

- 重新检查服务器地址、用户名和密码，注意大小写及前后空格。
- 确认 **群组名称** 和 **密钥** 与服务器端设置完全一致。
- 换用 Wi-Fi 或蜂窝网络后重试。
- 确认服务端支持 Cisco IPSec；如果管理员提供的是 **IKEv2** 参数，应在“类型”中选择 IKEv2，不能混用。
- 仍无法连接时，把 iPhone 显示的错误信息发给网络管理员，但不要发送密码或共享密钥。

## 删除配置

进入 **设置** → **通用** → **VPN 与设备管理** → **VPN**，点按配置旁的 ⓘ，再点按 **删除 VPN**。
